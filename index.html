How To Use

NOTE: Add Your All Task Fast Then Click On The Done Button

Step 1: Adding Tasks

To add a task, the user can enter a task in the input field and click the "Add Task" button. The entered task will be added to the task list.

Step 2: Editing Task Description

To edit the description of a task, the user can click on the task text. A modal box will open where the user can enter or modify the task description. Clicking the "Save" button will save the description, and clicking the "Close" button will close the modal box.

Step 3: Moving Tasks

To move a task to a different section, the user can drag and drop the task from one section to another. The app supports four sections: "Open," "In Progress," "Review," and "Done." The tasks will be visually moved to the corresponding sections as they are dragged and dropped.

Step 4: Managing Tasks

The user can manage their tasks by adding, editing, moving, and deleting them as needed. The app provides a simple and intuitive interface for organizing and tracking tasks. That's it! With these five steps, users can effectively use the task management app to stay organized and keep track of their tasks.
